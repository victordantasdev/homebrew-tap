# homebrew-tap

Homebrew tap for [victordantasdev](https://github.com/victordantasdev)'s tools.

## Usage

```sh
brew tap victordantasdev/tap
brew install <formula-name>
```

## Available formulas

- [`ghostty-config`](https://github.com/victordantasdev/ghostty-config) — Configure Ghostty shaders and themes with live preview.

  ```sh
  brew install victordantasdev/tap/ghostty-config
  ```

## Maintenance

Formulas in this tap are published automatically by [GoReleaser](https://goreleaser.com/) from the source repositories of each tool. Do not edit `Formula/*.rb` by hand — changes will be overwritten on the next release.
